## What "On Track" Looks Like

At any point in this project, Claude Code sessions should be able to answer:
- What are we building and why? → `docs/VISION.md`
- What does done look like right now? → `docs/MILESTONES.md`, current milestone
- What decisions have been made and why? → `docs/adr/`
- What does the system look like? → `docs/ARCHITECTURE.md`
- What's the next thing to build? → current milestone acceptance criteria

If any of those questions can't be answered from the docs, the docs need updating before more code gets written.
