# Telesis

**Development intelligence platform** — the control system around autonomous coding agents.

Telesis is the operating layer between the human who defines what to build and the agents who build it. It has memory. It maintains intent across sessions, PRs, milestones, and contributors.

Early development — MVP phase. See [docs/MILESTONES.md](docs/MILESTONES.md) for current progress.
